

Hudební interpreti a jimi vydaná alba - aplikace s WPF, Entity Framework 6 a databází.

Umožňuje autory a jimi vydaná alba do DB přidávat, editovat a odstraňovat.

Program správně funguje, pokud je nainstalována LocalDB SQL Server Express 2019, která je jako jediná
LocalDB součástí Visual Studia Community 2022. Více informací je v obrázcích Navod-1, Navod-2 a Screen.

- při ukládání nového nebo editaci uloženého autora/alba kontroluje vyplnění polí, zadání roku povolí
  pouze číslice v počtu 4 ks, rozsah 1900 až aktuální rok, vydání alba ne starší než začátek interpreta, ...
